PGP Whole Disk Encryption

If you are using PGP Whole Disk Encryption with Mac OS X, the usual install methods won't work, because they install rEFIt on the encrypted partition.

Reportedly, PGP Whole Disk Encryption uses an additional hidden HFS+ partition for bootstrapping, and manually installing rEFIt there works. Obviously, you won't have access to the encrypted partition if you use rEFIt to boot another operating system.
