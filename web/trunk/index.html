The rEFIt Project

rEFIt is a boot menu and maintenance toolkit for EFI-based machines like the Intel Macs. You can use it to boot multiple operating systems easily, including triple-boot setups with Boot Camp. It also provides an easy way to enter and explore the EFI pre-boot environment.

News

2006-06-10: There is now a StartupItem that keeps rEFIt active across Mac OS X updates and prevents problems with Safe Sleep.

2006-05-14: Version 0.7 was released. It includes a tool to synchronize the GPT and MBR partition tables and has extensive customization options.

2006-05-13: The Documentation is now available online.

Download

The current release is 0.7. It is available in various forms. The disk image distributions can be burned to CD. All three binary distributions can be used to install rEFIt on a USB memory stick or other disk drives.

  • rEFIt 0.7 (1.3M Mac disk image)
  • rEFIt 0.7 (1.3M ISO image)
  • rEFIt 0.7 (1.3M .tar.gz)
  • rEFIt 0.7 source code (75K .tar.gz)

You can browse the source code online through the ViewVC interface to the Subversion repository at SourceForge.

Resources

This project is hosted by SourceForge. You can use the trackers to submit bug reports, feature requests, and patches. The latest development source code is available from the Subversion repository. Other resources are listed on the project summary page.

To discuss rEFIt with other people, check out the OnMac Forums or the Dual Boot forum at the OSx86 Project.

Links

rEFIt was initially developed to allow easy dual booting of Linux and Mac OS X on the Intel iMacs. There are two sites with information about Linux on Intel Macs: mactel-linux.org and osxbook.com.

Initial hints for accessing the EFI firmware on Intel Macs came from the Nakfull Propaganda blog. The OSx86 Project is another valuable resource.

Some EFI links:

  • EFI home page at intel.com
  • The TianoCore project
